# CSS Flexbox Demonstration Website

A simple and responsive **CSS Flexbox Demo Website** built using **HTML5** and **CSS3**.  
This project demonstrates how to create modern flexible layouts using Flexbox.

---

## 📌 Project Overview

This website contains:

- Responsive Navigation Bar
- Hero/Header Section
- Features Cards Section
- Image Gallery
- Footer Section

The main purpose of this project is to understand and practice **CSS Flexbox properties** such as:

- `display: flex`
- `justify-content`
- `align-items`
- `flex-direction`
- `flex-wrap`
- `gap`
- `flex-grow`
- `flex-basis`
