# CSS Flexbox Demonstration Website

A responsive and modern web page created using **HTML5** and **CSS3** to showcase the power of **CSS Flexbox** for building flexible and responsive layouts.

## 🚀 Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action Button
* Feature Cards using Flexbox
* Image Gallery Layout
* Responsive Footer Section
* Hover Effects and Smooth Transitions
* Mobile-Friendly Design

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox)

## 🎯 Purpose

The goal of this project is to demonstrate how CSS Flexbox can be used to create clean, responsive, and visually appealing web layouts without relying on external frameworks.
